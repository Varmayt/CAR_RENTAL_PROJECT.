# Car Rental System 

##  Project Overview
This is a simple Java-based Car Rental Management System.
It allows users to rent cars, view available cars, and calculate rental costs.

## Technologies Used
- Java
- jdbc
- IntelliJ IDEA

## Project Structure
- Car.java
- Customer.java
- Rental.java
- CarRentalSystem.java
- Main.java
- TestConnection.java(for knowing the successful connection to DB)
  
NOTE : IMPORTANT FILE -- Easiest — Add JAR directly to IntelliJ project(Dependency)

Download this file directly (it supports PostgreSQL 18):
----downloaD LINK -> https://repo1.maven.org/maven2/org/postgresql/postgresql/42.7.4/postgresql-42.7.4.jar

Once downloaded, go to IntelliJ and:
Right-click your project name (Main) → Open Module Settings and select Modules → Dependencies → + (Add) → JARs or directories and Browse to the downloaded postgresql-42.7.4.jar file → click Apply → OK
You’ll now see the JAR listed under External Libraries (like your JDK ones).

##  How to Run
1. Open the project in IntelliJ IDEA  
2. Run the file `Main.java`  
3. Follow the console instructions
